FitFusion is a React.js workout generator application designed to help users create customized workout routines based on their specific goals, muscle groups, and workout environments.

The application leverages a predefined set of exercises, workout schemes, and tempos to generate tailored workout plans. Users can select individual muscle groups or predefined workout routines, and the application will filter and shuffle exercises to create a balanced and effective workout.

## Technologies Used

- **React.js**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
- **Vite**: A build tool that provides a fast development environment.
- **ESLint**: A tool for identifying and fixing problems in JavaScript code.

## Features

- **Custom Workout Plans**: Generate workout routines based on selected muscle groups.
- **Predefined Routines**: Choose from a set of predefined workout routines.
- **Exercise Filtering**: Filter exercises based on user preferences and goals.
- **Dynamic Exercise Shuffling**: Ensure variety in workouts by shuffling exercises.
- **Rep Counter**: Track the number of repetitions for each exercise.
- **User-Friendly Interface**: Easy-to-use interface for selecting muscle groups and workout preferences.

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/nasa-apod-react-app.git
   cd fitfusion
   ```

2. Install dependencies:

   ```node
   npm install
   ```

## Running the Application

1. Start the development server:

```node
npm run dev

```

2. Open your browser and navigate to
   `http://localhost:5173`
